# Vibecoding 模拟器

“什么叫今晚P0就要上线了，AI救我！！”  
—— 欢迎来到一个以开发者生涯为题材的轻量 Roguelike（肉鸽）网页小游戏。  
你将通过选卡构筑、扛住随机事故、维持项目完整度，看看这次到底是神级上线，还是史诗级复盘。

## 项目简介

- **类型**：单机网页小游戏（纯前端，无后端依赖）
- **核心循环**：选卡构筑 -> 阶段事件 -> 资源结算 -> 推进进度 -> 结局判定
- **主题风格**：Vibecoding、开发日常、线上事故、极限翻盘
- **运行方式**：直接打开网页即可游玩

## 玩法说明

1. 点击“开始一局”，深呼吸，假装一切都在掌控中
2. 从 8 张随机卡中挑 3 张作为开局构筑（命运从第一张卡就开始歪）
3. 每个阶段继续选卡并结算挑战（有时你在玩游戏，有时游戏在玩你）
4. 关注关键属性变化：
   - 古法编程（`craft`）
   - Vibe 共鸣（`vibe`）
   - 稳定性（`stability`）
   - Token 储备（`tokens`）
   - 热度（`hype`）
   - 项目完整度（`integrity`）
5. 任一资源崩盘（如 Token 或完整度归零）可能提前结束
6. 走完全部阶段触发最终结局和分数：荣耀上线 / 狼狈上岸 / 连夜复盘，随机其一
